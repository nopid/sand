# A Simple Sandpile in Rust + Pyxel

A sample Rust + Pyxel app showing a small pile of sand evolving according
to Abelian Sandpile rules [1].

Keyboard map:
 - SPACE: add more grains
 - Z: erase all sand
 - LEFT/RIGHT: modify update speed
